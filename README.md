# yogurt
SpringCloud 学习笔记项目，包括：
- eureka-server：Eureka 服务注册中心
- config-server：Spring Cloud Config 分布式配置中心
- gateway：Zuul服务网关
- role-ms：角色管理微服务
- user-ms：用户管理微服务
- common-dependency：各微服务的公共依赖
